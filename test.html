🎮 Maze Race Game - Functionality Test

How to Play:

  1. Click "Generate New Maze" to create a maze
  2. Click "Start Race!" to begin the race
  3. Use WASD or Arrow Keys to move your player (blue circle with 👤)
  4. The computer (pink circle with 🤖) will automatically find the optimal path
  5. First to reach the red end cell wins!
  6. Press R to restart after a game ends

Features Implemented:

  • ✅ Maze generation using recursive backtracking
  • ✅ DFS pathfinding algorithm for computer
  • ✅ Real-time user movement with WASD/Arrow keys
  • ✅ Computer AI that moves step-by-step
  • ✅ Win detection and celebration animations
  • ✅ Restart functionality
  • ✅ Visual distinction between players
